# +TITLE: A simple expert system in python
* Requirements
- python3.6+

* Usage
#+BEGIN_SRC bash
python3 main.py [filename]
#+END_SRC

* Grammar
- `+` -> binary `and`
- `|` -> binary `or`
- `^` -> binary `xor`
- `!` -> unary `not`
- `=>` -> implication, `a->b`: if a then b
- `<=>` -> bi-directional implication
- `=ARGS` -> fact list
- `?ARGS` -> hypothesis list

* Example
#+BEGIN_SRC bash
Type "quit" to exit the program.
#=> a+b=>c
#=> c=>d
#=> e=>a
#=> =be
#=> ?d
Rule d, True
#=>
#+END_SRC
